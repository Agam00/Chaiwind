Typography & Text

Controls for font sizing, coloring, weight, and general text aesthetics.

Class Pattern CSS Effect Example
chai-text-[color] color: [color] chai-text-#10b981, chai-text-white, chai-text-rgba(255,255,255,0.5)
chai-text-[size] font-size predefined scale chai-text-sm, chai-text-3xl, up to chai-text-7xl
chai-text-[align] text-align: [align] chai-text-center, chai-text-right
chai-font-[family] font-family: [family] chai-font-sans
chai-bold font-weight: bold chai-bold
chai-italic font-style: italic chai-italic
chai-tracking-[val] letter-spacing: [val]px chai-tracking-2
chai-leading-[val] line-height: [val] chai-leading-1.5

Spacing (Padding & Margin)

Numeric values are safely converted to pixel dimensions directly.

Class Pattern Target Property Supported Sides
chai-p-[val] Padding All Sides
chai-px-[val], chai-py-[val] Padding Left/Right (x) or Top/Bottom (y)
chai-pt-[val], pr, pb, pl Padding Top, Right, Bottom, Left respectively
chai-m-[val] Margin All Sides
chai-mx-[val], chai-my-[val] Margin Left/Right or Top/Bottom (e.g., chai-mx-auto)
chai-mt-[val], mr, mb, ml Margin Top, Right, Bottom, Left respectively

Layout & Sizing

Class Pattern CSS Effect Notes
chai-w-[val], chai-h-[val] width / height Accepts full (100%), screen (100vh/vw), or px value.
chai-max-w-[val], chai-min-h-[val] max-width / min-height chai-min-h-screen, chai-min-h-800
chai-flex, chai-grid display: flex / grid -
chai-absolute, chai-fixed position Also supports relative, sticky
chai-top-[val], bottom, left, right Placement coordinates chai-top-0, chai-inset-0
chai-z-[val] z-index: [val] chai-z-50

Premium Styling & Borders
